## 동적 변수
var로 선언

    var a = '스트링입니다.'
    var b = '3'
    var c = 3

동적언어 이므로 자료형 선언 불필요  
기본 자료형 과 객체(Object)

|기본 자료형| 설명 |
|------|--|
| Boolean   |논리적 자료형 true or false  |
|Null|객체값 없음|
|Undefined|값 할당하지 않음|
|Number|숫자형|
|String|문자형|
|Symbol|유일하고 변경 불가|
